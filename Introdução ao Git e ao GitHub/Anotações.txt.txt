O Git Bash é um terminal extendido para otimizar o uso do Git.

Comandos Básicos do Git:
git status = "Verificação de arquvios, juntamente com os não auterados ou confirmados"
git init = "Criação de um novo repositório" 
git clone = " Alocar na máquina um repositorio que esta em um servidor remoto"
git commit = "Confirmar alterações nos arquivos de sua máquina"
git add = "Adiciona ao controle de verão local os arquivos criado em sua máquina"
git config = "Definir valores de configurações especificas, como e-mail"
